# 在线外卖管理系统 java960

## 项目概述

在线外卖管理系统是基于B/S架构和MVC模式设计的，采用前后端分离的技术方案。该系统致力于为管理员和商家提供一个便捷、高效的管理平台。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK1.8
  - Maven
  - Mysql 5.7+
- **技术框架**: 
  - Java
  - Mysql
  - SSM（Spring、SpringMVC、MyBatis）
  - MyBatis-Plus
  - VUE
  - jQuery
  - HTML
- **数据库可视化工具**: Navicat
- **服务器**: SpringBoot 内嵌 Apache Tomcat

## 功能模块

### 系统角色

- 管理员
- 商家

### 功能列表

1. **通用功能**
   - 登录功能
   - 注册功能

2. **管理端功能**
   - 系统首页
   - 个人中心
   - 商家管理
   - 用户管理
   - 商品分类管理
   - 商品信息管理
   - 留言备注管理
   - 系统管理

3. **商家端功能**
   - 系统首页
   - 个人中心
   - 商品信息管理
   - 留言备注管理
   - 订单管理

## 运行环境

- 服务器端：建议配置为符合Java应用服务器运行的标准环境
- 客户端：支持现代浏览器，例如Chrome、Firefox等

## 核心亮点

- 前后端分离，易于维护和扩展
- 采用成熟的SSM框架和VUE.js，保障系统性能和开发效率
- 提供完整的商家与管理员功能模块，覆盖外卖业务全流程
- 支持多种数据库可视化工具，方便数据管理和维护
- 使用SpringBoot内嵌Tomcat，简化部署流程，降低环境搭建难度
